## Part B — Distribution Comparison

- Generated:
  - Normal distribution
  - Uniform distribution
- Visualized both distributions using histograms

Comparison:
- Shape: Normal is bell-shaped, Uniform is flat
- Spread: Uniform evenly distributed, Normal concentrated near mean
- Central tendency:
  - Normal centered around mean
  - Uniform evenly spread across range

Use cases:
- Normal → real-world natural data (heights, scores)
- Uniform → random sampling, equal probability events
